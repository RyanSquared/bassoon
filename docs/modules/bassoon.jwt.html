encode (data, table)
Encode data into a formatted JWT. Current supported formats are: ["HS256", "HS512", "none"]

Parameters:

  • data table JSON-serializable table
  • table opts
    • opts.secret_key: str = bassoon.util.random_key! - 24 bit random key
    • opts.algorithm: str = "HS512" - JWT algorithm
