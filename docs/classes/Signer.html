Class Signer

Implementation of hashing and signing short text, courtesy of luaossl

Usage:

    import Signer from require "bassoon"

Info:

  • Author: RyanSquared vandor2012@gmail.com

Methods

Signer\new (signing_key, opts)
Create a new Signer class object with stored secret_key and other data

Parameters:

  • signing_key string 24-byte unique-enforced key for signing text
  • opts table
    • opts.separator: str = "!" - separator for signed text
    • opts.digest_method: str = "sha512" - hashing algorithm

Usage:

    signer = Signer random_key!, separator: "."
print signer\sign "hello world"
Signer\sign (text)
Return a signed version of the text to send to clients

Parameters:

  • text string Text to sign. Should not include @separator

Usage:

    headers\set_cookie signer\sign "user=#{user}"
Signer\verify_text (Text, Signature)
Submethod used when verifying text without using @separator

Parameters:

  • Text text to be used for verification
  • Signature signature to match text
Signer\verify (signed_text)
Verify that a text matches the resulting signature

Parameters:

  • signed_text string Text with signature

    Signer\sign

Usage:

    cookie = headers\get_cookie_as_kv!
if signer\verify headers\get_cookie!
