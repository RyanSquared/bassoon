bassoon\new (signing_key, opts)
Create a new Signer class object with stored secret_key and other data

Parameters:

  • signing_key string 24-byte unique-enforced key for signing text
  • opts table
    • opts.separator: str = "!" - separator for signed text
    • opts.digest_method: hashing algorithm used when hashing text

Return a signed version of the text to send to clients
